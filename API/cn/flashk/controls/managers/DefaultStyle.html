graphicsDrawOffsetproperty 
public static var graphicsDrawOffset:Number = 0.5

对于圆角矩形，将此值设为0.5附近的值可以获得更好的反锯齿效果，并且在播放品质为中时边框线条不会模糊，对于直角矩形，在播放品质为低的情况下建议将此值设为0.
