Packagecn.flashk.controls
Classpublic class TileList
InheritanceTileList List UIComponent flash.display.Sprite

Language Version : ActionScript 3.0
Runtime Versions : AIR 1.0, Flash Player 9.0

TileList用来显示一个类似于系统资源管理器一样的缩略图列表，列表中的项目按照网格排列 默认显示缩略图的itemRender是cn.flashk.controls.support.TileListThumbnail
