Packagecn.flashk.controls
Classpublic class Tree
InheritanceTree List UIComponent flash.display.Sprite

Language Version : ActionScript 3.0
Runtime Versions : AIR 1.0, Flash Player 9.0

Tree 组件使用户可以查看排列为可扩展树的层次结构数据。树中的每个项目都可以是叶或分支。叶项目是树中的端点。分支项目可以包含叶或分支项目，也可以为空。 Tree有两种操作方式，批量操作和碎片单个操作。通过addItemChildIn之类的第三个参数指定，true为碎片操作，false为批量操作。删除操作 _tree.removeOneItem(_xml.item[0].item[0].item[0]);delete _xml.item[0].item[0].item[0];为碎片操作。 对于删除Tree子节点，先操作dataProvider的XML使用delete (XML) 运算符 ，然后调用updateWholeTreeNow()为批量操作。 对于复杂的Tree组合变换，可以对原始XML进行操作，然后调用updateWholeTreeNow()（或直接将XML重新赋值给dataProvider）即可,Tree会自动刷新。此时为批量操作。
