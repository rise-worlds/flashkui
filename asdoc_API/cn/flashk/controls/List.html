Packagecn.flashk.controls
Classpublic class List
InheritanceList UIComponent flash.display.Sprite
Subclasses DataGrid, TileList, Tree

Language Version : ActionScript 3.0
Runtime Versions : AIR 1.0, Flash Player 9.0

List 组件将显示基于列表的信息，并且是适合显示信息数组的理想选择。默认显示一列文字、图标（如果有）。
