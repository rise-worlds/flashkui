public static function registerLinkTip(display:DisplayObject, linkNameOrClass:*, initDataFunction:Function, ... args):void

对一个可交互的显示对象注册复杂提示

Parameters

display:DisplayObject — 可交互的显示对象
 
linkNameOrClass:* — 提示要使用的类名，库链接名，当需要显示Tip时，会new一个这样的实例
 
initDataFunction:Function — 当Tip即将显示时，将调用此函数初始化复杂tip的显示，此函数接收第一个参数为new linkName()的实例，后面参数为args的参数顺序（可选）
 
... args — 需要传递给initDataFunction的多个可选参数 Xtip.registerLinkTip(this,"com.dolo.ui.controls.Label",Xtip.InitFunction);
