Packagecn.flashk.controls
Classpublic class ToggleButton
InheritanceToggleButton Button GraphicSkinButton UIComponent flash.display.Sprite
Subclasses CheckBox, RadioButton

Language Version : ActionScript 3.0
Runtime Versions : AIR 1.0, Flash Player 9.0

ToggleButton 组件定义切换按钮。单击该按钮会在弹起状态和按下状态之间进行切换。如果在按钮处于弹起状态时单击该按钮，则它会切换到按下状态。必须再次单击该按钮才可将其切换回弹起状态。
